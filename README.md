## 構成

- `index.html` — 1ページ完結の静的サイト（ビルド不要）

## 公開（Netlify）

GitHub リポジトリと Netlify を連携した継続デプロイで公開。`main` への push で自動再デプロイされる。
`netlify.toml` により publish ディレクトリはリポジトリルート、ビルドコマンドなし。
